Front-end web developement, also known as client-side developement is the practice of producing HTML, CSS and JavaScript for a website or Web Application so that a user can see and interact with them directly. The challange associated with front end developement is that the tools and technique used to create the front end of a websites change constantly and so the developer needs to constantly be aware of how the field is developing.


The objective of designing a site is to ensure that when the users open up the site the see the information in a format that is easy to and relevent. This is further complicated by the fact that users now use a large variety of devices with varing sreen size and resolution thus forcing the degigner to take into consideration these aspects when desining the site. They need to ensure that their site comes up correctly in different browser (cross-browser), different operating systems (cross-plateform) and different devices (cross-devices), which requires carefull planning on the side of developer.
